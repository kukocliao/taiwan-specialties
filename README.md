# 🗺️ 台灣特產地圖大冒險 (Taiwan Specialties Map Adventure)

本專案是一款專為國小社會教材設計的遊戲化學習（Gamification）工具，旨在解決國小學童在背誦「台灣家鄉特產、自然成因、文化意涵與對應鄉鎮區」時的記憶痛點。

---

## 🎯 專案開發目的

家長為了幫助女兒克服國小社會科課本中「台灣特產與地理知識」繁雜難記的學習障礙，特別規劃了此互動地圖遊戲。透過地圖直覺點擊與趣味測驗，將生硬的課本知識轉化為好玩的遊戲素材，達到寓教於樂的效果。

### 📚 教材知識整理與分類系統
本遊戲完整梳理了國小教材中的核心特產知識，並將其進行結構化分類：
1. **自然資源類（成因）**：例如新竹因「九降風」而利於自然風乾製成彈性米粉；金山因「火山沙質土壤」種出鬆軟甜美的地瓜。
2. **人文傳統類（意涵）**：例如美濃油紙傘，「油紙」諧音「有子」，「傘」字字型內有四個人，象徵多子多孫與吉祥團圓。
3. **節慶與觀光推廣**：如南投魚池鄉的「世界茶業博覽會」（體驗揉茶聞茶香）、三義的「三義木雕藝術節」等。
4. **全台特產走透透（縣市與鄉鎮區精確對照）**：地圖精確對照全台 20 個行政區（包含外島金門、馬祖、澎湖），並細緻下探至課本考點的具體**鄉鎮區**（如大湖草莓、大溪豆干、林邊蓮霧等）。

---

## 🎮 遊戲功能特色

### 1. 📖 自由探索模式 (Study Mode)
- **直覺式互動地圖**：採用 D3.js 渲染高精度台灣行政區地圖，邊界清晰。點擊任一縣市即可於右側「探索筆記本」放大顯示其專屬特產。
- **鄉鎮區高亮標記**：特產詳情卡片中，特別以天藍色顯眼標籤展示**鄉鎮區**（如 `📍 三義`、`📍 大湖`），方便孩子第一時間連結縣市與在地鄉鎮。
- **特產照片**：每項特產附有實景照片，支援點擊全螢幕放大（Lightbox），並標示圖片來源。

### 2. 🏆 挑戰賽模式 (Quiz Mode)
- **多樣化三大題型**：
  - `🔍 產地尋寶`：詢問該縣市與鄉鎮區的特產為何（選項為特產名稱）。
  - `📌 鄉鎮對決`：詢問該特產屬於哪一個具體的鄉鎮區（選項為鄉鎮名稱）。
  - `🌧️ 自然/氣候成因`：結合自然地理因素詢問產地縣市（選項為縣市名稱）。
- **地圖點擊答題**：縣市類題型可直接點擊左側地圖作答，增強地理空間概念。
- **趣味回饋系統**：計時倒數、生命值、答對閃爍提示、專屬彈窗說明及過關紙花特效。

### 3. 📱 跨裝置支援
- 支援桌機、平板、手機操作。
- 手機採上下捲動布局（地圖固定於上方），平板與桌機為左右並排。

---

## 🛠️ 技術架構

- **前端**：純靜態 HTML5 / JavaScript / TailwindCSS CDN
- **地圖渲染**：D3.js v7，GeoJSON 資料已**嵌入 HTML 內部**（無需伺服器）
- **音效**：Web Audio API 純代碼合成，無需外部音檔
- **部署**：GitHub Pages 靜態託管

## 🚀 執行方式

### 本地直接開啟
由於 GeoJSON 已嵌入 HTML，**直接雙擊 `index.html`** 即可在瀏覽器中遊玩，不需要任何伺服器或安裝。
